# 📚 Bookstore SQL Analysis Project

## 📖 Project Overview

This project demonstrates SQL data analysis using a relational Bookstore database built in PostgreSQL. The database consists of three interconnected tables: **Books**, **Customers**, and **Orders**.

The project answers real-world business questions using SQL, including sales analysis, inventory tracking, customer insights, and book performance.

---

## 🎯 Objectives

- Analyze bookstore sales data
- Practice SQL querying techniques
- Perform business-oriented data analysis
- Demonstrate relational database concepts using PostgreSQL

---

## 🛠️ Technologies Used

- PostgreSQL
- pgAdmin 4
- SQL

---

## 🗂️ Database Schema

The project contains three tables:

### 📚 Books
- Book ID
- Title
- Author
- Genre
- Published Year
- Price
- Stock

### 👤 Customers
- Customer ID
- Customer Name
- City
- Country

### 🛒 Orders
- Order ID
- Customer ID
- Book ID
- Quantity
- Order Date

# 📈 Business Questions Solved

✔ Retrieve books by genre

✔ Find books published after a specific year

✔ List customers from a specific country

✔ Show orders placed within a given date

✔ Calculate average book price by genre

✔ Find the most frequently ordered book

✔ Show the Top 3 most expensive Fantasy books

✔ Calculate total quantity sold by each author

✔ Identify customers who placed multiple orders

✔ Calculate remaining stock after fulfilling all orders
